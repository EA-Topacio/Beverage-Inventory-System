A Business Intelligence and Inventory Management System developed using **Google Sheets**, **Google Apps Script**, and **JavaScript** to automate beverage inventory operations for banquet events.

The system streamlines banquet event order (BEO) management, beverage allocation, inventory tracking, stock movement approval, inventory reconciliation, and dashboard reporting within a centralized platform.

---

# 📌 Business Problem

Managing beverage inventory for banquet operations involves coordinating multiple events, beverage packages, stock movements, and inventory counts. Manual inventory tracking is often time-consuming and susceptible to human error, resulting in inaccurate stock records, delayed replenishment, and inefficient reporting.

This project addresses these challenges by developing an automated inventory management system that centralizes banquet event management, inventory transactions, and business intelligence reporting to improve operational efficiency and inventory accuracy.

---

# 🎯 Objectives

The project aimed to design and develop a centralized inventory management system capable of:

- Managing Banquet Event Orders (BEO)
- Recording beverage package allocations
- Tracking beverage consumption
- Managing stock movement requests
- Monitoring inventory levels
- Calculating inventory variance
- Providing interactive KPI dashboards for inventory reporting

---

# 📊 System Overview

The system consists of six integrated modules:

- Create BEO
- Manage BEO
- Stock Movement Requests
- Inventory Management
- Dashboard Reporting
- Navigation Menu

Each module automates a key business process while maintaining data consistency across the system.

# 🧠 System Features

## 🍷 Banquet Event Management

- Create new Banquet Event Orders
- Assign beverage packages
- Add additional beverages
- Search existing events
- Update beverage allocations
- Record actual beverage consumption
- Delete event records

---

## 📦 Inventory Management

The system automatically tracks:

- Opening Stock
- Stock Movement
- Expected Stock
- Monthly Stock Count
- Inventory Variance
- Recommended Restock Quantity

Inventory calculations are performed automatically using spreadsheet formulas and Apps Script automation.

---

## 🚚 Stock Movement Management

Supports inventory movement through an approval workflow.

Features include:

- Create movement requests
- Pending approval queue
- Approve requests
- Automatic inventory updates
- Complete movement history

# 🗄 Database Design

The system utilizes a relational database structure consisting of:

- BEVERAGE
- PACKAGE
- PACKAGE_BEVERAGE
- EVENT
- EVENT_PACKAGE
- EVENT_BEVERAGE
- STOCK_MOVEMENT
- STOCK_COUNT

The database design ensures consistent relationships between events, beverages, packages, and inventory records.
