# Beverage Inventory & Consumption System

## Overview

A Business Intelligence and Inventory Management System developed using **Google Sheets** and **Google Apps Script** to automate beverage inventory operations for banquet events.

The system streamlines banquet event order (BEO) management, inventory tracking, stock movement approval, inventory reconciliation, and dashboard reporting in a centralized platform.

## Features

### Banquet Event Order (BEO)

- Create new BEOs
- Allocate beverage packages
- Add additional beverages
- Search existing BEOs
- Update beverage allocations
- Record actual beverage consumption
- Delete BEO records

---

### Inventory Management

- Track opening stock
- Record stock movement
- Calculate expected inventory
- Perform monthly stock counts
- Calculate inventory variance
- Reset inventory for a new month

---

### Stock Movement Workflow

- Create stock movement requests
- View pending requests
- Approve requests
- Automatic inventory updates
- Movement log

---

### Business Intelligence Dashboard

#### KPI Cards

- Total Beverages
- Pending Requests
- Low Stock Items
- Inventory Value
- Event Consumption
- Inventory Variance
- Inventory Accuracy
- Total BEOs

#### Charts

- Top 10 Consumed Beverages
- Largest Inventory Discrepancies

#### Recommendations

- Automated Restock List

## Database Design

The system is built around a relational database structure consisting of:

- BEVERAGE
- EVENT
- EVENT_BEVERAGE
- EVENT_PACKAGE
- PACKAGE
- PACKAGE_BEVERAGE
- STOCK_COUNT
- STOCK_MOVEMENT
